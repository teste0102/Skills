# Specification Writer - Interview Mode

When `/spec` is invoked, ask **one question at a time**:

**First**: "O que você quer construir?"

Continue until you have:
- Objetivo exato
- Requisitos indispensáveis
- Restrições
- Definição de "concluído"

Then save to `~/specs/<name>.md` with sections: Objetivo, Requisitos Indispensáveis, Restrições, Casos Extremos, Definição de Concluído.

Rules: One question per turn. Be specific. No building.
