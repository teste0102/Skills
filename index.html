💡 Como Funciona?

Skills do Claude Code são carregadas automaticamente quando Claude detecta que são relevantes para sua tarefa. Você NÃO precisa digitar comandos especiais. Apenas descreva o que quer e Claude usará a skill certa!

📖 Como Usar (Exemplos)

"escreva uma spec para um app de tarefas" → espec
"construa conforme a spec" → build
"verifique se está conforme a spec" → review
"busca patinete no mercado livre" → buscar-ml
"pesquisa sobre inteligência artificial" → buscador
"analisa meu projeto e recomenda skills" → explorador
"cria uma skill para deploy automático" → criador
