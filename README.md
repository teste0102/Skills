# ⚡ sikavial-skills

Spec-driven development skills para Claude Code.

## Instalação

```bash
bash <(curl -s https://raw.githubusercontent.com/teste0102/Skills/main/install.sh)
```

## Como Usar

Skills são carregadas **automaticamente** pelo Claude Code. Não precisa digitar comandos especiais.

| Você diz... | Skill usada |
|---|---|
| "escreva uma spec para..." | espec |
| "construa conforme a spec" | build |
| "verifique a spec" | review |
| "itere até aprovar" | iterate |
| "busca X no mercado livre" | buscar-ml |
| "pesquisa sobre X" | buscador |
| "analisa meu projeto" | explorador |
| "cria uma skill para X" | criador |
| "revisa a árvore de skills" | curador |

## Skills (6)

| Nome | Descrição |
|---|---|
| espec | Entrevista para escrever specs |
| build | Constrói conforme a spec |
| review | Verifica compliance com spec |
| iterate | Loop automático build→review→fix |
| buscador | Busca na internet com fontes |
| buscar-ml | Busca no Mercado Livre |

## Agentes (4)

| Nome | Descrição |
|---|---|
| orquestrador | Roteia para agente correto |
| explorador | Recomenda skills para o projeto |
| criador | Gera novas skills automaticamente |
| curador | Mantém marketplace sincronizado |
